# 一、目标

### ① 这一步的最终目标只有一个：

> **用户登录成功后，Spring Security 能拿到：**
>
> - user_id
> - username
> - roles（ROLE_XXX）
> - permissions（system:user:view …）

并且能支持：

```java
@PreAuthorize("hasRole('ADMIN')")
@PreAuthorize("hasAuthority('system:user:view')")
```

⚠️ **注意**：

- 这一步 **不引入 JWT**
- 先用 Spring Security 的标准认证流程
- 权限从数据库加载（后面再换 Redis）

### 3️⃣ 查询用户角色（sys_user_role → sys_role）


示例 SQL（和你表结构完全一致）：

```sql
SELECT r.*
FROM sys_user_role ur
JOIN sys_role r ON ur.role_id = r.role_id
WHERE ur.user_id = #{userId}
  AND ur.status = 1
  AND ur.deleted = 0
  AND r.status = 1
  AND r.deleted = 0;
```

SQL（重点看条件）：

```sql
SELECT DISTINCT p.*
FROM sys_user_role ur
JOIN sys_role_permission rp ON ur.role_id = rp.role_id
JOIN sys_permission p ON rp.perm_id = p.perm_id
WHERE ur.user_id = #{userId}
  AND ur.status = 1 AND ur.deleted = 0
  AND rp.status = 1 AND rp.deleted = 0
  AND p.status = 1 AND p.deleted = 0;
```

# 五、第三步：组装 GrantedAuthority（非常关键）

### 1️⃣ 角色 → ROLE_XXX

```java
Set<GrantedAuthority> authorities = new HashSet<>();

for (SysRole role : roles) {
    authorities.add(
        new SimpleGrantedAuthority(role.getRoleCode())
    );
}
```

> ⚠️ 你的 `role_code` **必须是 `ROLE_XXX`**
> 这一点你前面已经设计对了

------

### 2️⃣ 权限 → perm_code

```java
for (SysPermission perm : permissions) {
    authorities.add(
        new SimpleGrantedAuthority(perm.getPermCode())
    );
}
```

✔ 最终 authorities 里同时有：

```text
ROLE_ADMIN
system:user:view
system:user:add
```

------

### 3️⃣ 返回 SecurityUser

```java
return new SecurityUser(

    user.getUsername(),
    user.getPassword(),
    user.getStatus() == 1,
    authorities
```

# 六、第四步：Spring Security 配置（最小可跑版）

```java
@Configuration
@EnableMethodSecurity
public class SecurityConfig {

    @Bean
    public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
        http
            .csrf(csrf -> csrf.disable())
            .authorizeHttpRequests(auth -> auth
                .requestMatchers("/login").permitAll()
                .anyRequest().authenticated()
            )
            .formLogin(form -> form.disable())
            .httpBasic(Customizer.withDefaults());

        return http.build();
    }

    @Bean
    public PasswordEncoder passwordEncoder() {
        return new BCryptPasswordEncoder();
    }
}
```
